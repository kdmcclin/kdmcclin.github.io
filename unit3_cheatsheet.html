Ruby Cheat Sheet

There are many different methods in Ruby to interact with arrays and hashes. Below are only handful. This work shouldn't be considered exhaustive but should be considered a work in progress as I will add to it as I find methods that I want to avoid searching for in the Ruby documentation.

  • Adding To
  • Deleting From
  • Searching

Adding To

There are simple methods for adding to both arrays and hashes. However, the methods are slightly different because hashes are composed of key with corresponding values, whereas arrays are just elements, albeit at a particular index.

Array

To add elements to an array, use either the #push method or double arrows.

array = [1, 2, 3, 4]
array.push(5)
array << 5
The result will be:
array = [1, 2, 3, 4, 5]

Hash

To add keys and values to a existing hash, enclose the new key in square brackets and set the brackets equal to the new value.

hash = {"green" => 1, "purple" => 2, "blue" => 3, "red" => 4}
hash["yellow"] = 5
The result will be:
hash = {"green" => 1, "purple" => 2, "blue" => 3, "red" => 4, "yellow" => 5}

Deleting From

The same #delete method can be called on both arrays and hashes to delete an element from an array or a key-value pair from a hash. Enclose the selected key or element within parentheses as in the following examples.

Array

array = [1, 2, 3, 4]
array.delete(1)
The result will be:
array = [2, 3, 4]

Hash

hash = {"green" => 1, "purple" => 2, "blue" => 3, "red" => 4}
hash.delete("green")
The result will be:
hash = {"purple" => 2, "blue" => 3, "red" => 4}

Searching

Arrays and hashes both have specific methods that allow you verify if something exists in that array or hash. For arrays you use #include? and for hashes you use #has_key? or #has_value?, depending on which one you are searching for.

Array

array = [1, 2, 3, 4]
array.include?(1)
The result will be:
=> true

Hash

hash = {"green" => 1, "purple" => 2, "blue" => 3, "red" => 4}
hash.has_key?("green")
The result will be:
=> true
hash.has_value?(1)
The result will be:
=> true
